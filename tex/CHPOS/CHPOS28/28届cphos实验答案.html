＊数值评分遵循以下标准：
\[ Y=y_{0}, \quad \text { Num }=A-B-X ; \Delta=0-\alpha-\beta \]
若考生给出 \(y\) 的有效位数为 \(A\) ，数值为 \(\left|y-y_{0}\right| \leq \alpha\) ，不扣分；有效位数为 \(B\) ，或 \(\left|y-y_{0}\right| \in(\alpha, \beta]\) ，或物理量少单位，扣除 \(50 \%\) 分数；有效位数为 \(A, B\) 以外的值，或 \(\left|y-y_{0}\right|>\beta\) ，扣除 \(100 \%\) 分数，该数据点得分扣完为止。计算过程中有效数字可以多（少）保留一位。没有给出Num的，表示与答案不同的有效位数扣除 \(\mathbf{1 0 0} \boldsymbol{\%}\) 分数；没有给出 \(\Delta\) 的，允许在末位出现 \(\mathbf{1}\) 的偏差，否则扣除 \(\mathbf{1 0 0} \%\) 分数。
